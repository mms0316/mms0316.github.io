Schematic tools

Material List

  • Counts materials of a schematic (.nbt, .schem, .litematic)
  • Breaks down materials into ingredients

Previewer

  • Renders simplified 3D model

Map Art Tools

Map Art Checker

  • Verifies if a starting location is valid for a map art generated by MapArtCraft
  • Sets slabs to top type
  • Converts Water to Waterlogged Oak Leaves, Glow Lichen to Verdant Froglight

Map Art Previewer

  • Converts back to source image
  • Lists materials used
